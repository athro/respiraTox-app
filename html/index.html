The respiraTox project
The respiraTox project is funded by the NC3Rs and suported by Shell. The project partners are Fraunhofer ITEM and the University of Birmingham.
Alert/Information
The respiraTox prediction service
The respiraTox prediction service has been implemented by Matthias Wehr and Andreas Karwath. The Github project can be found here: https://github.com/athro/respiraTox-app
The respiraTox Prediction Model
The respiraTox prediction model on pulmonary irritation has been trained using 1997 organic compounds. The model uses binary data on absence/occurrence of irritation; either sensory irritation or histopthalogical effects indicating tissue irritation or damage.
Information in irritation properties are retrieved from the following sources:

 Acute toxicity studies with inhalation exposure from the ECHA CHEM database - https://echa.europa.eu/ (source named: echa  acute tox),
 Repeated dose studies with inhalation exposure from the Fraunhofer RepDose® database -https://repdose.item.fraunhofer.de/ (source named: RepDose)
 Hazardous Substance Inventory from Toxnet - https://toxnet.nlm.nih.gov/newtoxnet/hsdb.htm (source named: HSDB)
 Existing datasets on Alarie assays and their RD50 values - http://www.pitt.edu/~rd50/ (source named: alarie assay)
 Irritation Endpoints  ECHA - https://echa.europa.eu/ (source named: echa  irritation)
 C&L Harmonized - ECHA - https://echa.europa.eu/de/regulations/clp/harmonised-classification-and-labelling, (source named: echa  C&L H)
 Acute Tox Inhalation Studies from the Fraunhofer AcuTox database (source named: AcuTox)

Information on structures are retrieved in the form of smile codes from:

1. Fraunhofer RepDose database
2. ChemIDplus1
3. Chemicalbook2
4. SmilesCas3

Using the smile codes we calculated physicochemical properties to be used by the training algorithm. The Tools for calculating these properties include CDK4, RDKit5, and Indiego6.

Also we used output of several QSAR models from EpiSuite7 and Vega Hub8 and structural descriptors in the form of Morgan Fingerprints9, also known as circular fingerprints (ECFP10). A more detailed description of all applied descriptors and models will be published.

In our 5-fold cross validation of trainings data we measured the models performance in area under ROC curve (AUC) and the current model resulted in an AUC of 0.71.
